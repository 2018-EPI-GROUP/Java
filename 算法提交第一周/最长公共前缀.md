## 最长公共前缀

编写一个函数来查找字符串数组中的最长公共前缀。

如果不存在公共前缀，返回空字符串 `""`。

**示例 1:**

```
输入: ["flower","flow","flight"]
输出: "fl"
```

**示例 2:**

```
输入: ["dog","racecar","car"]
输出: ""
解释: 输入不存在公共前缀。
```

**说明:** 所有输入只包含小写字母 `a-z` 。

```java
public static void main(String[] args) {
		int n;
		Scanner sc = new Scanner(System.in);
		n=sc.nextInt();			
		String[] S1= new String[n];				//数组n个字符的字符数组
		int i;
		for(i=0;i<S1.length&&S1[i]!=" ";i++) {
			S1[i]=sc.next();
		}
		String S2 = longestCommonPrefix(S1);
		System.out.println(S2);
	}
	   public static String longestCommonPrefix(String[] strs) {
	        int i,j,min;
	        int size = strs.length;
	        if(size!=0)								//如果字符数组中有字符，则执行以下代码
	        {
	        	min=strs[0].length();
		        for(i=1;i<size;i++) {				//寻找字符串中的最小长度
		        	if(strs[i].length()<min)
		        		min=strs[i].length();
		        }
		        char P1[][] = new char[size][];
		        for(i=0;i<size;i++) {				//将每个字符串存入字符型二维数组中
		        	P1[i]=strs[i].toCharArray();
		        }
		        int count=0,flag=0,flag1=0;
	        	for(j=0;j<min;j++) {				//每个字符串中每个字符遍历
	        		flag=0;
	        		for(i=0;i<size-1;i++)			//n个字符串同一列的字符进行对比
	        		{
	        			if(P1[i][j]==P1[i+1][j]) {
		        			flag++;
		        		}else						//如果找到不相同的字符，则推出循环
		        		{
		        			flag1=1;
		        			break;
		        		}
	        		}
	        		if(flag==size-1) {				//如果字符串中都有连续相同的字符，计数器加1
	        			count++;
	        		}
	        		if(flag1==1)					//如果找到不相同的字符，则推出循环
	        			break;
	        	}
	        	if(count==0)
	        		return "";
	        	else {
	        		StringBuilder str = new StringBuilder();
	        		for(i=0;i<count;i++) {
	        			str.append(P1[0][i]);
	        		}
	        		String result = str.toString();
	        		return result;
	        	}
	        	
		     }
	        else return "";					//如果字符串数组中没有字符串，则直接返回"";
	  }
}
```
