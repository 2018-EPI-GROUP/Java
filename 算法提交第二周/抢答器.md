## 抢答器

- 问题描述

  实现一个竞拍抢答程序：要求设置3个抢答者（3个线程），而后同时发出抢答指令，抢答成功者给出成功提示，为抢答成功者给出失败提示。

  ###### 代码实现

  第一位抢答者类

  ```java
  public class first extends Thread{
  	answer ae = new  answer();
  	public first(answer ae) {
  		this.ae=ae;
  	}
  	@Override
  	public void run() {
  		try {
  			Thread.sleep(10);
  		} catch (InterruptedException e) {
  			e.printStackTrace();
  		}
  		synchronized(ae) {
  			if(ae.status==false) {
  				System.out.println("1号抢到");
  				ae.status=true;
  			}else {
  				System.out.println("1号抢答失败");
  			}
  		}
  	}
  }
  
  ```

​	**同步类**

```java
public class answer {
	public boolean status = false;
	public void question() {
		System.out.println("请问，人有多少根头发！");
	}
}

```

​	**主程序**

```java
	public static void main(String[] args) {
		answer ae = new  answer();
		second one = new second(ae);	
		third two = new third(ae);
		first three = new first(ae);
		ae.question();
		three.start();
		one.start();
		two.start();
	}

```
