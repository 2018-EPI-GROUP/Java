## 纸牌游戏----小猫钓鱼

- **问题描述**

  将一副扑克牌平均分成两份，小哼和小哈各拿一份。小哼先出牌，然后小哈出的牌放在小哼刚打出的扑克牌上

继而小哼和小哈交替出牌。出牌时如果某人打出的牌与桌上的某张牌的牌面相同，即可将两张相同的牌及其中间所夹的牌全拿走，并以此放在自己手中牌的末尾。

- **解决思路**

  放牌用栈，出牌和取牌用队列

- **解决代码**

​        ==接口==

```java
package 扑克;

import java.util.Stack;

public interface seek {
	public abstract int giveCard(Stack<Integer> s); 
	public abstract void getCard(Stack<Integer> s,int mark);
}

```

 ==玩家类==

```java
package 扑克;

import java.util.Iterator;
import java.util.LinkedList;
import java.util.Queue;
import java.util.Stack;

public class Person implements seek{
	String name;
	Queue<Integer> card = new LinkedList<Integer>();
	Person(String name){
		this.name=name;
	}
	@Override
	public int giveCard(Stack<Integer> s) {
		return s.push(this.card.poll());
	}
	@Override
	public void getCard(Stack<Integer> s,int mark){
		Iterator<Integer> it = s.iterator();
		int flag =0;
		while(it.hasNext()) {
			if(it.next()==mark){
				flag=1;
				break;
			}
		}
		if(flag==0) {
			this.giveCard(s);			
		}
		if(flag==1){
			this.giveCard(s);
			this.card.offer(s.pop());
			while(s.peek()!=mark) {
				this.card.offer(s.pop());
			}
			this.card.offer(s.pop());
		}	
	}
}

```

==游戏进行==

```java
public static void main(String[] args) {
		final int MAX = 12;
		Scanner sc = new Scanner(System.in);
		Person p1 = new Person("小哈");
		Person p2 = new Person("小哼");
		Integer card[]= new Integer[MAX];
		for(int i=0;i<MAX;i++) {
			card[i]=sc.nextInt();
		}
		p1.card = deal(card,0);
		p2.card = deal(card,MAX/2);
		Stack<Integer> stack = new Stack<>();
		while(!p1.card.isEmpty()&&!p2.card.isEmpty()) {
			if(!p1.card.isEmpty()) {
				p1.getCard(stack, p1.card.peek());
				System.out.println("p1出牌:" + p1.card.peek());
			}else break;
			if(!p1.card.isEmpty()) {
				p2.getCard(stack, p2.card.peek());
				System.out.println("p2出牌:" + p2.card.peek());
			}else break;
			
		}
		if(p1.card.isEmpty()) {
			printMes(p1,p2);
		}else {
			printMes(p2,p1);
		}
			
		System.out.print("桌上牌是:");
		System.out.println(stack);
	}
	public static Queue<Integer> deal(Integer[] card,int start) {
		Queue<Integer> queue = new LinkedList<>();
		for(int i=0;i<card.length/2;i++) {
			queue.add(card[start++]);
		}
		return queue;
	}
	public static void printMes(Person p1,Person p2) {
		System.out.println(p1.name + "Win");
		System.out.println(p2.name + "当前手中的牌是:" + p2.card);
	}

```
