## 字母异位词

- **问题描述**

  给定两个字符串 s 和 t ，编写一个函数来判断 t 是否是 s 的字母异位词。

  ```c
  示例 1:
  
  输入: s = "anagram", t = "nagaram"
  输出: true
  示例 2:
  
  输入: s = "rat", t = "car"
  输出:false
  ```

 说明:假设字符串只包含小写字母。



- **代码演示**

```java
  public boolean isAnagram(String s, String t) {
       char ch1[] = s.toCharArray();	//将字符串s转换为ch1字符数组
	    char ch2[] = t.toCharArray();   //将字符串t转换为ch1字符数组
    	//将两个字符数组分别进行排序
	    Arrays.sort(ch1);	
	    Arrays.sort(ch2);
	    int i;
      	//如果s和t的长度不一样，直接返回false
	    if(ch1.length！=ch2.length)
	    	return false;
       //依次判断这两个排序好的字符串对应的字符是否相等
	    for(i=0;i<ch1.length;i++) {
            if(ch1[i]!=ch2[i])	//如果发现有不同的，直接返回false
                return false;
	    }
      //如果都相等，返回true
	    return true;
    }
```
