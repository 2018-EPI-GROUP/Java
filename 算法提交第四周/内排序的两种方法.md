## 内排序的两种方法

#### 冒泡排序

```c
void BubbleSort(int num[]) {
	int i, j;
	for (i = 0; i < MAX-1 ; i++) {		//总共寻找n-1趟
        //从第一项开始，依次比较向量元素值的大小，将最大的元素排到“最后”
		for (j = 1; j < MAX-i; j++) {	
			if (num[j] > num[j + 1])	//如果后一个元素比前一个元素小，则交换这两个元素
			{
				num[0] = num[j];
				num[j] = num[j+1];
				num[j+1] = num[0];
			}
		}
	}
}
```

#### 直接选择排序

```c
void SelectSort(int num[]) {
	int i, j, min;
	for (i = 1; i < MAX; i++) {
		min = i;		//将每次遍历的第一个数字设为最小值
		for (j = i;j <= MAX; j++) {	//查找从i以后的最小的数
			if (num[min] > num[j])
				min = j;		//记录最小数的下标
		}
		/*交换每次开始数和最小数*/
		num[0] = num[i];
		num[i] = num[min];
		num[min] = num[0];
	}
}
```
